# Cálculo de despesas de viagem (C – Unidade 2)

Programa interativo em C que:
1) Lê distância (km)  
2) Lê preço do litro (R$)  
3) Considera consumo fixo de 10 km/L  
4) Calcula e exibe o gasto total formatado em duas casas decimais (`%.2f`)

## Como compilar e executar (GCC)
```bash
gcc despesa_viagem.c -o despesa
./despesa
