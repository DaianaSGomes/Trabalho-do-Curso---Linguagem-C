# Trabalho-do-Curso---Linguagem-C
Criação de um programa interativo em C para cálculo de despesas de viagem. Este trabalho tem como objetivo aplicar os conceitos básicos da Unidade 2: como criar variáveis, ler o que o usuário digita e mostrar um resultado na tela. É o ponto de partida para entender como os programas processam dados reais.
